# Observations custom field: Lot ID

Adds a structured "Lot ID" field to Observations so the lot reference is
queryable data as well as a title prefix. OpenLot itself links via the title
(works with zero configuration); this field is belt-and-braces for Procore
reporting and future structured linking.

Layout: `Field Label, Field Type, Tool, Required, Description`. Confirm the
current custom-field import format with Procore before submitting; custom
fields can also be created manually in Company Admin in under a minute.
