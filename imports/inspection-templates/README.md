# Civil ITP inspection templates

CSV layout: one row per checklist item with section grouping — the common
shape for Procore inspection template bulk imports. Columns:

`Template Name, Section, Item Number, Item Description, Response Type, Spec Reference`

Adjust to the current Procore import template before submitting (see
../README.md). The item text encodes the checkpoint type where relevant:
(H) hold point, (W) witness point, (S) surveillance.
